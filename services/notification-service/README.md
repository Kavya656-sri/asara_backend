# Notification Service

**Port:** 3008  
**Database:** `fsm_notifications`  
**Migration Phase:** Phase 2

## Responsibility

Firebase push notifications, in-app notification inbox

## Getting Started

1. Copy relevant files from `Easy_FSM_backend/src/` (see docs/migration/MONOLITH_ROUTE_MAPPING.md)
2. Update database config to use `fsm_notifications`
3. Install dependencies: `npm install`
4. Run: `npm run dev`

See [Migration Guide](../../docs/migration/MICROSERVICE_MIGRATION_GUIDE.md) for full details.
