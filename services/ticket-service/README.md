# Ticket Service

**Port:** 3005  
**Database:** `fsm_tickets`  
**Migration Phase:** Phase 4

## Responsibility

Tickets, Freshdesk sync, ticket workflow, ticket-task operations

## Getting Started

1. Copy relevant files from `Easy_FSM_backend/src/` (see docs/migration/MONOLITH_ROUTE_MAPPING.md)
2. Update database config to use `fsm_tickets`
3. Install dependencies: `npm install`
4. Run: `npm run dev`

See [Migration Guide](../../docs/migration/MICROSERVICE_MIGRATION_GUIDE.md) for full details.
