# Attendance Service

**Port:** 3006  
**Database:** `fsm_attendance`  
**Migration Phase:** Phase 5

## Responsibility

Daily attendance, config, force checkout cron, attendance logs

## Getting Started

1. Copy relevant files from `Easy_FSM_backend/src/` (see docs/migration/MONOLITH_ROUTE_MAPPING.md)
2. Update database config to use `fsm_attendance`
3. Install dependencies: `npm install`
4. Run: `npm run dev`

See [Migration Guide](../../docs/migration/MICROSERVICE_MIGRATION_GUIDE.md) for full details.
